# gin-vben-admin

这是一个基于 Gin 的后端接口项目，用于支持 `vben-admin` 前端框架。

## 项目结构

- `main.go`: 入口文件
- `api/`: 接口层
- `model/`: 数据模型
- `service/`: 业务逻辑层
- `router/`: 路由配置
- `middleware/`: 中间件
- `utils/`: 工具函数
- `config/`: 配置文件
